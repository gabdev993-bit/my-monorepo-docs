📚 My Monorepo 프로젝트 문서

이 사이트는 My Monorepo 프로젝트의 모든 문서를 포함합니다.

📋 Planning (계획 및 설계)

  • 프로젝트 개요 - 프로젝트 기본 정보 및 기술 스택
  • 아키텍처 설계 - 시스템 아키텍처 및 데이터 플로우
  • 로드맵 - 개발 단계별 계획
  • 요구사항 명세서 - 기능 및 비기능 요구사항

🛠️ Development (개발 환경)

  • 환경 설정 가이드 - 전체 개발 환경 설정
  • Xcode 설정 - iOS 개발 환경
  • Android 설정 - Android 개발 환경

📱 Mobile (모바일 앱)

  • 인증 설정 - Supabase + Google OAuth 설정
  • 인증 기능 - 인증 시스템 사용법
  • 네비게이션 - 라우팅 및 네비게이션

🔄 Shared (공유 문서)

  • 코딩 컨벤션 - TypeScript, React Native 코딩 규칙
  • 문제 해결 - 일반적인 문제 및 해결 방법

🚀 빠른 시작

1. 개발 환경 설정

# 의존성 설치
pnpm install

# 모바일 앱 개발 서버 시작
cd apps/mobile
pnpm dev

2. 주요 명령어

# iOS 시뮬레이터 실행
pnpm ios

# Android 에뮬레이터 실행
pnpm android

# 빌드
eas build --platform all

3. 문서 읽기 순서

  1. 코딩 컨벤션 - 핵심 개발 규칙 숙지
  2. 프로젝트 개요 - 프로젝트 이해
  3. 환경 설정 가이드 - 개발 환경 구축
  4. 인증 설정 - 인증 시스템 설정
